# PsycheESE
User guide for using SeisSol to study impact seismology on small bodies. Example files and data from the PsychESE project. 

This work was created in partial fulfillment of Arizona State University Capstone Course “SES 410/411″. The work is a result of the Psyche Student Collaborations component of NASA’s Psyche Mission (https://psyche.ssl.berkeley.edu). “Psyche: A Journey to a Metal World” [Contract number NNM16AA09C] is part of the NASA Discovery Program mission to solar system targets.
